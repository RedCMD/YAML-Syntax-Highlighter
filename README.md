# YAML Ain't Markup Language syntax

## Features

- Syntax highlighting
- Bracket matching
- YAML versions [1.2](https://yaml.org/spec/1.2.2), [1.1](https://yaml.org/spec/1.1/) and [1.0](https://yaml.org/spec/1.0/)
